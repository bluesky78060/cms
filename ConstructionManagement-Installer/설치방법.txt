================================================================================
                    🏗️ 건설업 관리 시스템 - 설치 방법 
                           ZIP 패키지 버전 v1.0
================================================================================

🚀 빠른 설치 (3단계):

  1단계: 이 ZIP 파일을 원하는 위치에 압축 해제
  
  2단계: "install.bat" 파일을 마우스 우클릭
         → "관리자 권한으로 실행" 선택
  
  3단계: 화면 안내를 따라 진행 (자동 완료)
         → 5-10분 후 설치 완료!

================================================================================

📋 포함된 파일들:

  🔧 설치 스크립트:
    • install.bat           - 자동 설치 (메인)
    • uninstall.bat         - 완전 제거
    • package-creator.bat   - 재배포용 패키지 생성
  
  📖 가이드 문서:
    • windows-deployment-guide.html - 20슬라이드 PPT 가이드
    • README.txt                   - 상세 설명서
    • 설치-가이드.md               - 마크다운 가이드
    • 설치방법.txt                 - 이 파일
  
  📁 프로그램 파일:
    • src/                  - React 소스코드
    • public/              - 정적 파일들
    • package.json         - 의존성 정보
    • package-lock.json    - 버전 잠금
    • craco.config.js      - 빌드 설정

================================================================================

⚠️ 중요사항:

  • 반드시 "관리자 권한으로 실행"해야 합니다
  • 인터넷 연결이 필요합니다 (패키지 다운로드용)
  • Windows 10/11 64bit에서만 동작합니다
  • 바이러스 백신이 설치를 차단할 수 있습니다

================================================================================

✅ 설치 완료 후:

  바탕화면에 다음 아이콘들이 생성됩니다:
    🟢 건설업관리시스템  - 시스템 시작
    🔴 시스템중지        - 시스템 종료  
    📖 설치가이드        - 사용 설명서

  웹 브라우저에서 http://localhost:3000 접속하여 사용하세요!

================================================================================
