⚙️ 기술 사양 및 시스템 요구사항

🛠️ 기술 스택 (Technology Stack)

Frontend: React 18.2.0

UI Framework: Tailwind CSS 3.2.7

상태 관리: React Context API

PDF 생성: jsPDF + AutoTable

Excel 처리: SheetJS (xlsx) 0.18.5

데이터 저장: Browser localStorage

배포: GitHub Pages

빌드 도구: CRACO (Create React App Config)

📱 지원 브라우저 및 환경

브라우저 최소 버전 권장 버전 특별 요구사항
Google Chrome 70+ 최신 버전 JavaScript 활성화 필수
Mozilla Firefox 65+ 최신 버전 localStorage 사용 허용
Safari 12+ 최신 버전 쿠키 및 사이트 데이터 허용
Microsoft Edge 79+ 최신 버전 팝업 차단 해제 권장

💻 권장 시스템 사양

  • 화면 해상도: 1024×768 이상
  • RAM: 4GB 이상
  • 저장 공간: 100MB (브라우저 캐시)
  • 인터넷 연결: 초기 로드시에만 필요

⚠️ 주의사항

  • 브라우저 시크릿 모드에서는 데이터 유지 불가
  • 브라우저 캐시 삭제 시 데이터 손실
  • 정기적인 Excel 백업 필수

🔒 도장 관리 시스템

📷 지원 이미지 규격

항목 권장 사양 지원 범위 비고
파일 형식 PNG (투명배경) PNG, JPG, JPEG 투명 배경 PNG 권장
파일 크기 1MB 이하 5MB 이하 로딩 속도 고려
이미지 크기 300×300px 100×100px ~ 1000×1000px 정사각형 권장
해상도 300 DPI 72 DPI 이상 인쇄 품질 고려

🛠️ 도장 업로드 프로세스

1단계: 이미지 선택

파일 선택 버튼 클릭 또는 드래그 앤 드롭

2단계: 자동 검증

파일 형식, 크기, 해상도 자동 확인

3단계: 미리보기

업로드된 이미지 실시간 미리보기 제공

4단계: 저장 완료

localStorage에 Base64 형태로 영구 저장

✅ 도장 최적화 가이드

  • 스캔 시 300DPI 이상 사용
  • 배경 제거 후 PNG 저장
  • 도장 테두리만 남기고 여백 제거
  • 명도/대비 조정으로 선명도 향상

⚠️ 주의사항

  • 개인 도장의 무단 사용 금지
  • 법인 도장 사용 권한 확인
  • 이미지 파일 별도 백업 권장
  • 브라우저 변경 시 재업로드 필요

3
작업 항목 추가: "+" 버튼으로 다중 작업 항목 입력
4
가격 계산: 단가 × 수량 자동 계산 및 총액 표시
5
일괄 저장: 모든 항목을 동일한 조건으로 한 번에 등록

📁 백업 데이터 구조

JSON 백업 파일 구성

{
  "timestamp": "2024-09-11T15:30:00.000Z",
  "version": "2.0",
  "data": {
    "COMPANY_INFO": { ... },      // 업체 정보
    "CLIENTS": [ ... ],           // 건축주 데이터
    "WORK_ITEMS": [ ... ],        // 작업 항목
    "ESTIMATES": [ ... ],         // 견적서
    "INVOICES": [ ... ],          // 청구서
    "STAMP_IMAGE": "data:image/png;base64,..."  // 도장 이미지
  },
  "statistics": {
    "totalClients": 15,
    "totalWorkItems": 128,
    "totalInvoices": 45
  }
}
