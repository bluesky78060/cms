🚀 빠른 설치 (권장)
================================================================================

1. 이 폴더의 "install.bat" 파일을 마우스 우클릭
2. "관리자 권한으로 실행" 선택
3. 화면의 안내를 따라 진행
4. 설치 완료 후 바탕화면의 아이콘으로 시스템 실행

⏱️ 설치 시간: 약 5-10분 (인터넷 속도에 따라 차이)

================================================================================
📋 설치 과정 상세
================================================================================

🔍 1단계: 시스템 요구사항 자동 확인
  • Windows 10/11 버전 확인
  • 메모리 4GB 이상 확인
  • 디스크 여유공간 2GB 이상 확인

🔧 2단계: Node.js 자동 설치/확인
  • 설치되지 않은 경우 자동으로 다운로드 및 설치
  • LTS 버전 (v18.x) 설치
  • npm 패키지 매니저 확인

📁 3단계: 프로젝트 파일 설정
  • C:\ConstructionManagement 폴더에 설치
  • 모든 프로젝트 파일 복사
  • 권한 설정 및 폴더 구조 생성

📦 4단계: 의존성 패키지 설치
  • npm install 자동 실행
  • 프로덕션 빌드 생성 (npm run build)
  • Express 웹서버 설정

⚙️ 5단계: 시스템 서비스 설정
  • 웹서버 자동 시작 스크립트 생성
  • Windows 서비스 등록 (선택사항)
  • 포트 3000 설정

🔗 6단계: 바탕화면 바로가기 생성
  • 건설업관리시스템.lnk - 시스템 시작
  • 시스템중지.lnk - 시스템 종료
  • 설치가이드.lnk - 사용 설명서

🎯 설치 후 사용법
================================================================================

시스템 시작:
  1. 바탕화면의 "건설업관리시스템" 아이콘 더블클릭
  2. 웹 브라우저가 자동으로 열림 (http://localhost:3000)
  3. 시스템 사용 시작

시스템 종료:
  1. 바탕화면의 "시스템중지" 아이콘 더블클릭
  2. 모든 Node.js 프로세스 자동 종료

네트워크 공유 (사무실 내):
  1. 시스템 시작 후
  2. 다른 컴퓨터에서 http://[서버IP]:3000 접속
  3. IP 주소는 cmd → ipconfig 명령으로 확인

================================================================================
🔧 문제 해결
================================================================================

❌ "관리자 권한이 필요합니다" 오류:
  → install.bat를 우클릭 → "관리자 권한으로 실행" 선택

❌ "Node.js를 인식할 수 없습니다" 오류:
  → 컴퓨터 재시작 후 다시 시도
  → 또는 수동으로 https://nodejs.org에서 Node.js 다운로드

❌ "포트 3000이 이미 사용 중" 오류:
  → 다른 프로그램이 같은 포트를 사용 중
  → 컴퓨터 재시작 후 다시 시도

❌ "npm install 실패" 오류:
  → 인터넷 연결 확인
  → 바이러스 백신 프로그램 일시 비활성화
  → 방화벽 설정 확인

❌ 브라우저에서 페이지가 열리지 않음:
  → Windows 방화벽에서 Node.js 허용
  → http://localhost:3000 직접 입력
  → 다른 브라우저로 시도

🗑️ 완전 제거 방법
================================================================================

자동 제거 (권장):
  1. uninstall.bat를 관리자 권한으로 실행
  2. 화면 안내에 따라 진행
  3. 모든 파일과 설정이 자동으로 제거됨

수동 제거:
  1. 바탕화면 바로가기 삭제
  2. C:\ConstructionManagement 폴더 삭제
  3. 제어판에서 Node.js 제거 (필요시)
  4. Windows 서비스 "건설업관리시스템" 제거
