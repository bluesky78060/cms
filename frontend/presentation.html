메인 화면 구성

📸 메인 화면 스크린샷
- 네비게이션 바
- 대시보드 영역
- 메뉴 구성

주요 메뉴 구성

  • 환경설정 - 업체 정보, 시스템 설정
  • 건축주 관리 - 고객 및 작업장 정보
  • 작업 항목 - 작업 목록 및 진행상황
  • 견적서 - 견적서 작성 및 관리
  • 청구서 - 청구서 발행 및 관리

환경설정 - 업체 정보 관리

📸 환경설정 - 업체정보 화면
- 업체명, 사업자등록번호
- 대표자명, 연락처
- 주소, 계좌정보

📝 설정 가능 항목

  • 기본 정보: 업체명, 사업자등록번호, 대표자명
  • 연락 정보: 전화번호, 이메일, 사업장 주소
  • 결제 정보: 계좌번호, 예금주명
  • 수정 기능: 실시간 수정 및 저장

💡 사용 팁

설정한 정보는 모든 견적서와 청구서에 자동으로 포함됩니다. 정확한 정보 입력이 중요합니다.

환경설정 - 도장 관리

📸 도장 관리 화면
- 도장 이미지 업로드
- 미리보기
- 저장소 정보

🔧 도장 관리 기능

  • 이미지 업로드: PNG, JPG 파일 지원 (최대 5MB)
  • 미리보기: 업로드된 도장 이미지 확인
  • 영구 저장: 브라우저를 닫아도 이미지 유지
  • 저장소 모니터링: 사용량 실시간 확인

✅ 권장사항

  • 투명 배경의 선명한 이미지 사용
  • 정사각형 비율 권장 (1:1)
  • 파일 크기는 1MB 이하 권장

건축주 관리

📸 건축주 관리 화면
- 건축주 목록
- 검색 및 필터
- 추가/수정 버튼

👥 건축주 관리 기능

  • 건축주 등록: 이름, 연락처, 주소 정보
  • 작업장 관리: 여러 작업장 등록 가능
  • 프로젝트 연결: 건축주별 프로젝트 관리
  • Excel 연동: 대량 데이터 가져오기/내보내기

🔍 검색 및 필터링

  • 건축주명으로 빠른 검색
  • 프로젝트 상태별 필터링
  • 청구 금액 및 미수금 현황

작업 항목 관리

📸 작업 항목 화면
- 작업 목록
- 상태 관리
- 일괄 등록 기능

⚒️ 작업 항목 기능

  • 작업 등록: 작업명, 카테고리, 가격, 수량
  • 상태 관리: 예정 → 진행중 → 완료
  • 일괄 등록: 동일 건축주의 여러 작업 한번에 등록
  • 가격 관리: 기본 단가 설정 및 조정

📊 진행 상황 추적

  • 작업별 진행 상태 색상 표시
  • 완료율 및 예상 수익 계산
  • 지연 작업 알림 기능

견적서 관리

📸 견적서 관리 화면
- 견적서 목록
- 상태 추적
- PDF 출력

📋 견적서 작성 프로세스

1건축주 및 작업장 선택

2작업 항목 추가 및 단가 설정

3견적서 제목 및 유효기간 설정

4PDF 생성 및 발송

5승인 시 작업 항목으로 자동 전환

📈 견적서 상태 관리

  • 검토중: 고객 검토 대기
  • 승인됨: 견적 승인 완료
  • 거절됨: 견적 거절
  • 작업 전환됨: 실제 작업으로 전환 완료

청구서 관리

📸 청구서 관리 화면
- 청구서 목록
- 생성 과정
- PDF 미리보기

💰 청구서 발행 프로세스

1완료된 작업 항목 확인

2청구할 작업 선택 (체크박스)

3청구서 정보 입력

4PDF 생성 (도장 자동 포함)

5발송 및 상태 관리

🖨️ PDF 출력 특징

  • 업체 정보 자동 포함
  • 도장 이미지 자동 삽입
  • 한국어 금액 표기 (예: 금 오백만원정)
  • 전문적인 청구서 양식

Excel 활용 기능

📸 Excel 가져오기/내보내기 화면
- 템플릿 다운로드
- 파일 업로드
- 데이터 매핑

📊 Excel 연동 기능

  • 템플릿 다운로드: 표준 양식 제공
  • 대량 가져오기: 기존 Excel 데이터 일괄 등록
  • 데이터 내보내기: 현재 데이터를 Excel로 백업
  • 형식 검증: 데이터 형식 자동 확인

📝 지원되는 데이터 유형

건축주 데이터

이름, 연락처, 주소, 작업장 정보

작업 항목

작업명, 카테고리, 단가, 수량

견적서/청구서

상세 내역 및 금액 정보

주요 사용 시나리오

🔄 시나리오 1: 신규 프로젝트 진행

1환경설정에서 업체 정보 및 도장 등록

2새 건축주 등록 및 작업장 정보 입력

3견적서 작성 및 발송

4견적 승인 후 작업 항목으로 전환

5작업 완료 후 청구서 발행

📄 시나리오 2: 기존 데이터 활용

1Excel 템플릿 다운로드

2기존 데이터 템플릿에 입력

3시스템으로 데이터 가져오기

4검토 및 수정

5정상 운영 시작

문제해결 가이드

🔧 일반적인 문제 해결

데이터가 사라졌어요

  • 브라우저 캐시 확인
  • 같은 브라우저 사용 확인
  • Excel 백업 파일 활용

도장 이미지가 안 보여요

  • 파일 크기 확인 (5MB 이하)
  • 이미지 형식 확인 (PNG/JPG)
  • 브라우저 새로고침

PDF가 생성되지 않아요

  • 팝업 차단 해제
  • 브라우저 업데이트
  • Javascript 활성화 확인

Excel 가져오기 오류

  • 템플릿 양식 준수
  • 필수 필드 입력 확인
  • 특수문자 제거

💻 권장 환경

  • 브라우저: Chrome, Firefox, Safari 최신 버전
  • 화면 해상도: 1024x768 이상
  • 네트워크: 오프라인 사용 가능
