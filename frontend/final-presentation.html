📝 설정 가능 항목

  • 기본 정보: 업체명, 사업자등록번호, 대표자명

🔧 도장 관리 기능

  • 이미지 업로드: PNG, JPG 파일 지원 (최대 5MB)

✅ 권장사항

  • 투명 배경의 선명한 이미지 사용
  • 정사각형 비율 권장 (1:1)
  • 파일 크기는 1MB 이하 권장

문제해결 가이드

🔧 일반적인 문제 해결

데이터가 사라졌어요

  • 브라우저 캐시 확인
  • 같은 브라우저 사용 확인
  • Excel 백업 파일 활용

도장 이미지가 안 보여요

  • 파일 크기 확인 (5MB 이하)
  • 이미지 형식 확인 (PNG/JPG)
  • 브라우저 새로고침

PDF가 생성되지 않아요

  • 팝업 차단 해제
  • 브라우저 업데이트
  • Javascript 활성화 확인

Excel 가져오기 오류

  • 템플릿 양식 준수
  • 필수 필드 입력 확인
  • 특수문자 제거

💻 권장 환경

  • 브라우저: Chrome, Firefox, Safari 최신 버전
  • 화면 해상도: 1024x768 이상
  • 네트워크: 오프라인 사용 가능 (localStorage 기반)

데이터 백업 및 보안

💾 데이터 백업 방법

정기적 Excel 내보내기

월 1회 이상 모든 데이터를 Excel로 내보내기하여 안전한 위치에 보관

중요 파일 별도 보관

PDF 청구서, 견적서 등 중요 문서는 별도 폴더에 체계적으로 보관

🔒 보안 고려사항

  • 로컬 저장: 모든 데이터가 사용자 컴퓨터에만 저장되어 외부 유출 위험 없음
  • 개인정보 보호: 건축주 연락처 등 개인정보 관리 주의
  • 정기적 업데이트: 브라우저 및 보안 프로그램 최신 버전 유지

⚠️ 주의사항

  • 브라우저 캐시 삭제 시 데이터 손실 가능성
  • 다른 컴퓨터에서 사용 시 데이터 동기화 불가
  • 하드웨어 고장에 대비한 정기적 백업 필수
